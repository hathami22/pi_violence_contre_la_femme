# Violence Against Women - Data Analysis Project

## Overview

This project focuses on analyzing and understanding patterns related to violence against women through data science and machine learning techniques. The repository contains Jupyter notebooks that provide comprehensive analysis, data exploration, and predictive modeling.

## Project Description

The primary goal of this project is to:
- Analyze datasets related to violence against women
- Identify patterns and risk factors
- Develop predictive models for early intervention
- Provide insights for policy and decision-making
- Support evidence-based awareness and prevention initiatives

## Getting Started

### Prerequisites

- Python 3.7 or higher
- Jupyter Notebook or JupyterLab
- Required Python packages (see below)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hathami22/pi_violence_contre_la_femme.git
cd pi_violence_contre_la_femme
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

## Data Analysis Workflow

The notebooks follow a typical data science workflow:

1. **Data Loading & Exploration** - Initial data inspection and summary statistics
2. **Data Cleaning & Preprocessing** - Handling missing values and data transformations
3. **Exploratory Data Analysis (EDA)** - Statistical analysis and visualizations
4. **Feature Engineering** - Creating meaningful features for modeling
5. **Model Development** - Building and training predictive models
6. **Evaluation & Interpretation** - Assessing model performance and extracting insights
